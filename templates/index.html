Technicalities

This project is a web app using Flask, PostgreSQL, and Gunicorn, with web hosting provided by Heroku. The front-end code uses HTML and CSS. Underneath the hood, the back-end is written in Python, using the Flask API for web application functions. The food and diary databases are hosted on Heroku via the PostgreSQL relational database management system. The web app's Python processes run on Gunicorn, a Web Server Gateway Interface (WSGI) HTTP server. Version control was organized through Git. Code from the latest version is available on Github.
